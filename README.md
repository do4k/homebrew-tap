# do4k Homebrew Tap

Homebrew formulae and casks maintained by [do4k](https://github.com/do4k).

## Installation

```bash
brew tap do4k/tap
```

## Available Casks

### unnaturalscrollwheels-do4k

Fork of [UnnaturalScrollWheels](https://github.com/ther0n/UnnaturalScrollWheels) — a tool to invert scroll direction for physical scroll wheels.

```bash
brew install --cask unnaturalscrollwheels-do4k
```

> **Note:** This conflicts with the original `unnaturalscrollwheels` cask. Uninstall it first if you have it installed.
